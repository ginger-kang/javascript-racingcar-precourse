# 🏎️ 자동차 경주 게임

## 🧑‍💻 구현 기능

- [x] 자동차들의 이름과 이동 횟수 입력 받기
- [x] 잘못된 입력일 경우 메시지와 함께 재입력 받기
  - 자동차들의 입력을 쉼표(,)로 구분하지 않음
  - 자동차의 이름이 5자 초과
  - 자동차를 한 대 이하로 입력
- [x] 입력받은 자동차들의 객체 생성
- [x] 자동차들이 전진할지 그대로 있을지 결정
  - 0~9 사이의 랜덤 값 중 4 이상일 경우 전진, 3 이하일 경우 정지
- [x] 주어진 횟수 만큼 자동차들을 전진 또는 정지
- [ ] 경주 결과를 보여주는 기능
  - 자동차들의 이름과 전진 횟수
- [ ] 최종 우승자를 보여주는 기능
  - 우승자가 여러명일 경우 쉼표(,)로 구분
